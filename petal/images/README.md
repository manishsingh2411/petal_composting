# Images Folder

Place your images here with the following names:

1. **compost-hands.jpg** - Image for "Hands holding rich, dark compost" (Hero section, first image)
2. **potted-plant.jpg** - Image for "A healthy, thriving plant in a pot" (Hero section, second image)

## Image Requirements:
- Recommended format: JPG or PNG
- Recommended size: 1080px width minimum
- Aspect ratio: 3:2 (width:height)
- File names must match exactly: `compost-hands.jpg` and `potted-plant.jpg`

## How to add images:
1. Save your images in this folder
2. Name them exactly as shown above
3. The website will automatically display them
